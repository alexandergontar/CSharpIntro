## Задача
* Написать программу, которая из имеющегося массива строк формирует массив из строк длиною не более 3-х символов. 

---
## Oписание решения.
1. Генерируется массив строк разной длины.
2. Выводится полученный массив. 
3. Определяется число строк в нем не более 3-х символов. 
4. Создается новый массив с таким количеством строк.
5. Строки исходного массива не более 3-х символов переписываются в новый массив. 
6. Выводится новый массив.
7. 
 прим: коллекции не используются
